# RaspberryInstallation
To install on a RaspberryPi for Escape Rush
fresh Raspbian-full
add EscapeRush user with sudo group
remove user pi

raspi-config : - boot into cli, no login
               - activate ssh
               
               
aptitude update && full-upgrade
omxplayer
library wiringPi
library libpam-systemd
arduino
avrdude ?
